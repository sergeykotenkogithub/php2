Функционал сайта:

- Асинхроное добавление товара в корзину
- Пагинация
- Отображение количество товаров в корзине
- Оформление товара
- Авторизация\Регистрация пользователя, есть возможность запомнить пользователя
- Страница с заказами пользователя
- Страница админ со всеми заказми всех пользователей
- Асинхронный скролинг с подгрузкой заказов на странице админ
- Возможность изменять статус любого пользователя
